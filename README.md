## 개요📗
‘타봤슈’는 카이스트 교내 약 8개의 자전거 정류장 정보를 보여주는 앱입니다. 회원가입, 로그인을 통해 앱을 사용하고 사용자 개인의 위치 정보 동의를 통해 주변에 가장 가까운 자전거 정류장의 정보를 제공합니다. 언제든 탈출할 수 있게 돕겠습니다.

## 개발 스택
- ### 개발 언어
  - Flutter / Dart
  - Flask / Python
  - MySQL

- ### IDE
  - Android Studio
  - VSC
